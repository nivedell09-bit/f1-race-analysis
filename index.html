F1 Race Performance Analysis using Machine Learning

Developed by Nivedhitha K

Project Overview

This project analyzes Formula 1 Bahrain Grand Prix 2024 race data using Python, FastF1, Machine Learning, and Data Visualization techniques.

Objective

Predict Formula 1 lap times using a Random Forest Regression model and identify key performance factors affecting race outcomes.

Tools Used

  • Python
  • Pandas
  • FastF1
  • Matplotlib
  • Seaborn
  • Scikit-Learn

Results

Model Performance

MAE: 0.289 seconds

R² Score: 0.986

Feature Importance

SpeedST: 72.95%

TyreLife: 17.83%

LapNumber: 5.02%

Project Visualizations

Conclusion

Speed Trap Speed and Tyre Life were the most influential factors affecting lap time performance. The Random Forest model achieved high predictive accuracy with an R² score of 0.986.
